Instrucciones de Importación

Tipos de importación:

  • 1
    Incremental (Recomendado)
    Agrega nuevos registros y actualiza existentes. Ideal para actualizaciones diarias o semanales.
  • 2
    Reemplazo Total
    Elimina todos los registros y carga los nuevos. Usar solo para carga inicial o correcciones masivas.
Tip: Puedes subir datos de cualquier período (día, semana, mes, año). El sistema detecta duplicados por código y los actualiza.
